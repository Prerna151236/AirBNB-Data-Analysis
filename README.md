🎯 Project Objectives

Clean and preprocess raw Airbnb dataset

Perform Exploratory Data Analysis (EDA)

Identify key patterns in pricing, revenue, and reviews

Build an interactive Power BI dashboard

Generate actionable insights for business decision-making

🗂 Dataset Description

The dataset contains information about Airbnb listings including:

Listing ID

Neighborhood

Room Type

Property Type

Price

Number of Reviews

Review Scores

Estimated Revenue

Availability

After data cleaning, the final dataset was used to build the dashboard.

🧹 Data Cleaning Steps

Data cleaning was performed using Python (Pandas).

Key steps included:

Removing unnecessary columns

Handling missing values

Converting data types

Removing duplicate records

Standardizing numerical fields

Exporting a clean dataset for analysis

📊 Exploratory Data Analysis (EDA)

EDA was performed to understand patterns in the dataset.

Key analyses included:

Price distribution analysis

Revenue estimation trends

Listing distribution by neighborhood

Room type distribution

Review activity analysis

🔍 Key Insights

Entire home/apartment listings dominate the market, representing the majority of listings.

Sydney has the highest concentration of Airbnb listings, indicating strong demand.

Listings with more reviews tend to have higher occupancy and better ratings.

Pricing plays an important role in determining revenue performance.
